# Unity - Fruit Picking Game

This is a block-style fruit picking game made with Unity. It features complete game logic and a high-score system.

## Gameplay Instructions

- Use the **A** and **D** keys to rotate the player's direction.
- Use the **W** and **S** keys to move forward and backward.
- When the player touches a red ball, they lose one life.
- The game ends when all lives are depleted.

## How to Run

### Play Online
Sorry, the web version is not yet deployed. The author is working on it.

### Run from Source Code
1. Download the entire repository.
2. Download and open [Unity Hub](https://unity.cn/releases).
3. Use Unity Hub to import the project folder and open it with the matching Unity version **6000.0.22f1c1**.

### Play Offline (Standalone Build)
1. Go to the [Releases](https://github.com/XuanEXE-system666/Unity-FruitFallingGame/releases) page of this repository.
2. Download the version for your operating system.
3. Extract the downloaded archive using an archive tool.
4. Run the corresponding executable file.

## How to Build

If you wish to build this project, be sure to use Unity **6000.0.22f1c1**'s Build Profile feature.

### Build Steps
1. Configure the scene list in the Build Profile.
2. Select and configure the target platform.
3. Click **Build**.
