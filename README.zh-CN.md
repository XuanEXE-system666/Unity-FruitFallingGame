# Unity-捡水果游戏

这是一个用 unity 制作的方块捡水果游戏。具有完整的游戏逻辑和最高分系统等。

## 玩法说明

- 使用a和d键旋转玩家朝向
- 使用w和s键前进和后退
- 当碰到红色球时玩家减少一条命
- 当命数耗尽时游戏结束

## 如何运行

### 在线运行
抱歉，当前网页版暂未部署，作者正在努力中

### 离线代码
1. 下载整个仓库。
2. 下载并打开[unity Hub](https://unity.cn/releases)
3. 使用Hub导入整个项目文件夹并匹配对应版本unity 6000.0.22f1c1打开。=

### 离线游玩
1. 前往本项目的[Release](https://github.com/XuanEXE-system666/Unity-FruitFallingGame/releases)界面
2. 下载对应你系统的版本
3. 使用解压缩软件解压下载的文件压缩包
4. 点击对应的程序文件运行

## 如何打包
如果你希望打包这个项目，务必使用Unity 6000.0.22f1c1的build profile功能

### 打包步骤
1. 在Build profile中配置场景列表
2. 点击对应的平台进行配置
3. 点击构建
